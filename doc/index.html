POST /playlists/:playlist_id/tracks

This service adds a track to an existing playlist.

  • curl -d 'track[title]=song1&track[album_title]=album1&track[artist_name]=artist1&track[rdio_id]=t123' http://localhost:9292/playlists/1/tracks

Input parameters description

Namespace: track

Required Params

  • title of type string The title of the track. 
  • album_title of type string The title of the album to which the track belongs. 
  • artist_name of type string The name of the track's artist. 
  • rdio_id of type string The Rdio ID of the track. 

Required Params

  • playlist_id of type integer The ID of the playlist to which the track belongs. 

Response description

Properties:
  • title of type string The title of the track.
  • album_title of type string The title of the album to which the track belongs.
  • artist_name of type string The name of the track's artist.
  • rdio_id of type string The Rdio ID of the track.

Response example

{"title":"string","album_title":"string","artist_name":"string","rdio_id":"string"}

DELETE /playlists/:playlist_id/tracks/:id

This service removes a track from an existing playlist.

  • curl -X DELETE http://localhost:9292/playlists/1/tracks/20

Input parameters description

Required Params

  • playlist_id of type integer The ID of the playlist to which the track belongs. 
  • id of type integer The ID of the track to be deleted. 

Response description

Properties:
  • title of type string The title of the track.
  • album_title of type string The title of the album to which the track belongs.
  • artist_name of type string The name of the track's artist.
  • rdio_id of type string The Rdio ID of the track.

Response example

{"title":"string","album_title":"string","artist_name":"string","rdio_id":"string"}

POST /playlists

This service accepts a list of musical artists, and then builds 15 track playlist with similarities to the provided artists.

  • curl -d 'artists=Weezer, Black Keys', http://localhost:9292/playlists

Input parameters description

Optional Params

  • artists of type array An list artists to use for track recommendations. 

Response description

Properties:
  • id of type integer The ID of the track that has been deleted.
  • tracks of type Array
    Properties of each array item:
    • title of type string The title of the track.
    • album_title of type string The title of the album to which the track belongs.
    • artist_name of type string The name of the track's artist.
    • rdio_id of type string The Rdio ID of the track.

Response example

{"id":"integer"}

GET /playlists/:id

This service returns a playlist.

  • curl 'http://localhost:9292/playlists/1'

Input parameters description

Required Params

  • id of type integer The ID of the playlist to which the track belongs. 

Response description

Properties:
  • id of type integer The ID of the playlist that was fetched.
  • tracks of type Array
    Properties of each array item:
    • title of type string The title of the track.
    • album_title of type string The title of the album to which the track belongs.
    • artist_name of type string The name of the track's artist.
    • rdio_id of type string The Rdio ID of the track.

Response example

{"id":"integer"}
